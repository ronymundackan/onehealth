# OneHealth

OneHealth is a comprehensive web application designed to streamline healthcare management for patients, doctors, and hospitals. It provides a centralized platform for managing appointments, accessing medical records, and facilitating communication between patients and healthcare providers.

## Features

**For Patients:**

* **User-friendly dashboard:** View and manage appointments, access medical records, and update personal information.
* **Appointment scheduling:** Easily schedule appointments with doctors and hospitals.
* **Medical record access:** Securely access and view personal medical records.
* **Communication tools:** Communicate with healthcare providers through secure messaging.

**For Doctors:**

* **Efficient appointment management:** Manage appointments, view patient schedules, and update availability.
* **Access patient records:** Access and update patient medical records with appropriate permissions.
* **Collaboration tools:** Collaborate with other healthcare providers on patient care.

**For Hospitals:**

* **Centralized patient management:** Manage patient records, appointments, and billing information.
* **Doctor and staff management:** Manage doctor and staff profiles, schedules, and permissions.
* **Reporting and analytics:** Generate reports and analyze data to improve healthcare services.

## Technologies Used

* **Frontend:** React, Lucide React (for icons)
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Authentication:** JWT (JSON Web Tokens)

## Installation and Setup

1. **Clone the repository:** `git clone https://github.com/ronymundackan/onehealth.git`
2. **Install dependencies (Frontend):** `cd onehealth/frontend && npm install`
3. **Install dependencies (Backend):** `cd onehealth/backend && npm install`
4. **Start the backend server:** `cd onehealth/backend && npm run dev`
5. **Start the frontend development server:** `cd onehealth && npm start`
